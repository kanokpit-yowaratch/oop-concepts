## Description

A simple OOP Concepts written in TypeScript.

## Install

```bash
npm install
```

## Build

```bash
npm run build
```

## Publish on local

```bash
npm link
```

## Link from other app to package
```bash
npm link @kanokpit.y/oop-concepts

# unlink from app to package
npm unlink @kanokpit.y/oop-concepts
```

## Unlink from other app to package
```bash
npm unlink @kanokpit.y/oop-concepts
```
